Comandos GitHub

  • git init: Inicializar repositório de estrutura
  • git clone: Clona de forma identica o repositorio alvo
  • git add. ou git add : Adiciona arquivos especifico ou todos sem especificação.
  • git commit -m 'mensagem de commit': salva suas alterações e devine a mensagem de ação realizada.
  • git push: Publica alterações no servidor, seja local ou remoto
  • git pull: Obtém atualizações do seu repositorio local, conforme o repositorio remoto
  • git merge: Reune as branch modificadas
  • git status: Exibe atual status do projecto, alterações e modificações realizadas do repositorio
  • git log: Exibe autor de alterações e o momento a qual foi alterado
  • git branch : Cria ramificações idempendente da branch master antes de subir o arquivo para produção e alteração

    • Visite: https://git-scm.com/docs
